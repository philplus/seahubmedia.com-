# Projects

This folder holds per-project context so we can resume quickly without relying on chat scrollback.

## How to use
- Each project has its own folder under `projects/<project-slug>/`.
- Put stable facts in `CONTEXT.md`.
- Log decisions/changes in `DECISIONS.md` (append-only, dated).
- Keep reusable prompts/templates in `PROMPTS.md`.
- Keep outputs in `outputs/`.
